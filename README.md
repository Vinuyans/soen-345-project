# Ticket Reservation App


## How to run
- Open in android studio
- Make sure the `google-services.json` is in the `app` folder.
- Start an emulator in the Device Manager tab
- Click Run App in the top right corner of Android Studio

Alternatively there is an app apk provided that can be installed on a Android device
